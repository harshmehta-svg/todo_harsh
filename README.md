## Overview
This project is a comprehensive Todo List application that allows users to manage their tasks effectively. It features core functionalities such as adding, completing, undoing, and deleting tasks. The application ensures data persistence by saving all tasks to the browser's local storage.

The repository contains two distinct implementations:
1.  A simple, lightweight version built with vanilla HTML, CSS, and JavaScript located in the public directory.
2.  A more modern, component-based version built using React, located in the src directory.
