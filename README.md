## Overview
This project is a Todo List application designed to help users manage their daily tasks. It includes core functionalities such as adding, completing, undoing, and deleting tasks. The application state, including the list of tasks, is persisted in the browser's local storage, ensuring that user data is saved across sessions.

The repository contains two distinct implementations: a simple, lightweight version built with vanilla HTML, CSS, and JavaScript located in the root directory, and a more feature-rich version built with React, utilizing the TodoList.js component.

## Technology Stack
- Frontend: ReactJS
- Backend: Node.js (not used in this version)
- UI Framework: None
- State Management: React Context API
