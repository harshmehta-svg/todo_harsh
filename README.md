## Overview
This repository contains a simple, client-side Todo List application and a React AI agent dashboard. The project allows users to manage their daily tasks by adding, completing, undoing, and deleting items. Additionally, the React AI agent dashboard enables users to engage in conversations with multiple threads, maintain their own message history, and switch between conversations. All tasks and conversation histories are persisted in the browser's local storage, ensuring data is saved between sessions.

The repository contains two distinct implementations:
1.  A core version built with vanilla HTML, CSS, and JavaScript, located in the root directory.
2.  A more advanced version built with React, located in the `src` directory, which includes features from the Todo List and the AI agent dashboard.

## Technology Stack
- Frontend: React.js
- Backend: Not required (client-side application)
- Database: Local Storage (browser-based)
