## Overview
This project is a Todo List application that allows users to manage their daily tasks. It features functionality to add, complete, undo, and delete tasks. The application state, including the list of todos and user login status, is persisted in the browser's local storage, ensuring data is saved between sessions.

The repository contains both a simple vanilla JavaScript implementation and a more advanced version built with React, which includes user authentication and a dark/light mode theme switcher. The code also includes a GitHub Personal Access Token for authenticated API requests.

## Technology Stack
- Frontend: React
- Backend: Node.js
- API: REST API
- State Management: Local Storage
- Styling: CSS
