## Overview
This project is a comprehensive Todo List application that allows users to manage their daily tasks effectively. It features core functionalities such as adding, completing, undoing, and deleting tasks. The application state is persisted in the browser's local storage, ensuring that the user's data is saved across sessions.

The repository contains multiple implementations, including a vanilla JavaScript version and several React-based components, suggesting an ongoing evolution of the codebase. It also incorporates features like user authentication, with a new signup page allowing users to create an account.

## Technology Stack
* Frontend: React
* Backend: Node.js
* Database: In-memory storage (for local storage of todo items)
* Authentication: Custom authentication service using Node.js
