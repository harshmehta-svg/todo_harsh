## Overview
This project is a simple Todo List application with a chat UI built with front-end technologies. It allows users to manage their tasks by adding, completing, undoing, and deleting them. The application also includes a chat UI where the user can type a message, send it to an AI API, and display the response in a chat bubble layout. The application persists all tasks in the browser's local storage, ensuring that the user's todo list is saved between sessions. The repository contains multiple implementations, including a vanilla JavaScript version and several React components, which also explore features like user authentication and a dark mode theme.

## Technology Stack
- **Language:** JavaScript
- **Front-end Framework:** React
- **Styles:** CSS/SCSS
- **Services:** Node.js and Express.js
- **API:** OpenAPI and HuggingFace API for chat UI functionality
