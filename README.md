## Overview
This project is a Todo List application that demonstrates task management functionalities. It appears to contain multiple implementations: a simple, vanilla JavaScript version at the root level and more complex, component-based versions using React within the `src` directory.

The application allows users to add, complete, undo, and delete tasks. All tasks are persisted in the browser's local storage, ensuring that the user's todo list is saved between sessions. The React implementation also includes user authentication and a new user list functionality.

## Technology Stack
- Frontend: React, JavaScript, HTML5, CSS3
- Backend: Node.js
- Storage: Browser's Local Storage
- Authentication: Custom Authentication using Node.js
