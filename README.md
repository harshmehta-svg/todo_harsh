## Overview
This repository contains a simple, client-side Todo List application. It allows users to manage their tasks by adding, completing, undoing, and deleting them. The application persists all tasks in the browser's local storage, ensuring that the user's data is saved between sessions.

The repository appears to contain two distinct implementations:
1.  A straightforward vanilla JavaScript version located in the root directory.
2.  A more advanced version built with React, located in the `src` directory, which includes features like component-based architecture and state management.

## Technology Stack
* Frontend: React
* Backend: Node.js
* Database: Local Storage
