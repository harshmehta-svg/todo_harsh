## Overview
This project is a Todo List application that allows users to manage their tasks. The repository contains multiple implementations, including a simple vanilla JavaScript version and a more complex version using React. Key functionalities include adding, completing, undoing, and deleting tasks. The application persists all task data in the browser's local storage, ensuring that the user's todo list is saved between sessions. It also includes features like user authentication, a dark/light mode theme switcher, and a new login page.

## Technology Stack
- Frontend: React
- Backend: Node.js
- Database: Local Storage (browser)
