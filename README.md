## Overview
This repository contains a simple yet functional Home design application. The project appears to have multiple implementations: a straightforward version built with vanilla HTML, CSS, and JavaScript located in the root directory, and a more advanced version using React located within the `src` directory.

The application allows users to design their ideal home by customizing various aspects such as layout, furniture, and decorations. All selections are persisted in the browser's local storage, ensuring that the user's home design layout is saved between sessions.

## Technology Stack
- Frontend: React
- Backend: Not required for this project
- Database: Local Storage (using browser's native API)
