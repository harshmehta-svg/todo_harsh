## Overview
This repository contains a feature-rich Todo List application designed to help users manage their daily tasks. The project utilizes React as the primary frontend framework, with a responsive Dashboard layout featuring a sidebar, header, and main content area.

## Technology Stack
- Frontend: React
- CSS Framework: Tailwind CSS
- JavaScript Library: Node.js
- State Management: Local Storage
