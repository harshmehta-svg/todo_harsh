# Project README

## Overview
This is a [nodejs/react](https://reactjs.org/) project that provides a simple Todo List application and Code Assistant feature.

## Features

### Todo List
A Todo List application that allows users to create, read, update and delete tasks.

### Code Assistant
A React component, CodeAssistant, that enables users to type code-related questions. It sends the question to an AI API and displays the AI's formatted code response with syntax highlighting using react-syntax-highlighter.
