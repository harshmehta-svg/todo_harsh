## Overview
This is a simple Todo List application built with front-end technologies, including HTML5, CSS3, and JavaScript. The application allows users to add, complete, and delete tasks.

## Technology Stack
- **HTML5**: For the structure and layout of the application
- **CSS3**: For styling and visual design
- **JavaScript**: For dynamic functionality and interactivity
- **React**: For building the user interface
- **TypeScript**: For static typing and better code maintainability
- **Jest**: For unit testing
- **Webpack**: For bundling and module management
